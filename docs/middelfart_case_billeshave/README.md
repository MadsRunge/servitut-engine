# Middelfart Case Billeshave

Denne mappe samler de originale PDF'er i projektet, som matcher Middelfart-sagen fra den tidligere blandede case.

## Indhold

- `Tinglysningsattest.pdf` er den originale attest for sagen
- `akter/` indeholder de tilhoerende akter, som matcher attesten

## Sagsspor

Dokumenterne peger paa ejendomme og matrikler omkring:

- `Ny Billeshavevej 4, 5500 Middelfart`
- `Rojle By, Strib-Rojleskov`
- `Billeshave Hgd., Strib-Rojleskov`

## Formål

Mappen er lavet som et rent testsaet, saa du kan koere OCR og extraction kun paa Middelfart-sagen uden at blande den med Aalborg-dokumenterne.

## Bemærkning

Denne mappe er afledt af den blandede case:

- `storage/cases/case-5831c642`

De dokumenter, der matcher `Gasvaerksvej 28, 9000 Aalborg`, er derfor bevidst ikke med her.
